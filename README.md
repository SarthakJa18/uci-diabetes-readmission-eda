#  UCI Diabetes Readmission EDA

## 📌 Problem Statement

The goal of this project is to perform **Exploratory Data Analysis (EDA)** on the **UCI Diabetes 130-US Hospitals Dataset** and understand the dataset before applying machine learning models for **patient readmission prediction**.

## 📂 Dataset Source

**UCI Machine Learning Repository**

https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008

## 📊 Key Observations

- Dataset contains **100,000+ patient records**
- Most columns have **no missing values**
- "max_glu_serum" and "A1Cresult" contain significant missing data
- Dataset contains both **categorical** and **numerical** features
- "readmitted" can be used as the **target variable**
